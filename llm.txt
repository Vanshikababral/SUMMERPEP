----Prompts---- 
df - A prompt is the instruction or input that you give to an LLM so it knows what task to perform.
Think of it as asking a question or giving directions to a very knowledgeable assistant.

Example - 1."What is Artificial Intelligence?"
          2."Summarize the following paragraph in 50 words."

2. Multi Msg Prompts 
eg. - conversation with a chatbot 
    (i) Messages(manual) - (uses system , human , AI messags template from langchain)
        1. System Message
            The System Message gives the AI its role, behavior, and instructions.
            It tells the AI:
                Who it is
                How it should behave
                What it should and shouldn't do
        2. Human Message
            This is the user's question.
            Every time the user types something, it becomes a HumanMessage.
        3. AI Message
            This represents the assistant's previous reply.
            Normally, you don't create it yourself for a single-turn conversation.
            It's mainly used when continuing a conversation.
        ****
        ChatPromptTemplate
            "from langchain_core.prompts import ChatPromptTemplate"
                This creates prompts containing
                System messages
                Human messages
            A ChatPromptTemplate is a LangChain class used to create chat-style prompts for chat models like:
            Claude
            ChatGPT
            Gemini
            Llama Chat models
            Instead of writing one long prompt, it lets you organize the conversation into multiple messages, such as:
            System Message
            Human Message
            AI Message (optional)
            Think of it as a template for an entire conversation, not just a single block of text.

What are we doing?
Pipeline: Input → Prompt → Model → Parser → Output
We first provide an input to the application.
That input is converted into a prompt (either a Single Message Prompt or a Multi-Message Prompt).
The prompt is then sent to the LLM (Model).
The model generates a response.
The response is passed to an Output Parser, which formats or extracts the required information.
Finally, the parsed result is displayed as the output.

Here are the steps in a short and easy-to-remember format:

(i)Import the required libraries.
(ii)Create the Prompt Template (ChatPromptTemplate).
(iii)Create the Model (ChatGoogleGenerativeAI).
(iv)Create the Output Parser (StrOutputParser).
(v)Invoke the Prompt using chat_prompt.invoke() to generate the final prompt.
(vi)Invoke the Model using model.invoke() to get the model's response (AIMessage).
(vii)Invoke the Parser using parser.invoke() to extract the text from the model's response.
(viii)Print the final output.
